# 변수와 상수

### var vs const vs let

- 변수: 바뀔 수 있는 값(let, var)
- 상수: 값을 지정하면 변경 할 수 없음 (const)
- var: 변수 선언을 중복으로 할 수 있음(이젠 사용하지 않음)

## null vs undefined

- null : 값이 존재 하지 않음
- undefined: 아직 정해지지 않음
