# Arrow Function

- ES6에 도입
- 익명함수를 간단하게 만들 수 있다.

```jsx
const add = (a, b) => {
  return a + b;
  }
  // const add = (a, b) => a + b; 도 같은 의미

  const sum = add(1, 2);
  console.log(sum);
```
